## 基础知识
- 启动参数`-classpath`指定执行的`类加载路径`
- 程序中`System.getProperty("java.class.path")`可以取到当前的程序的classpath


## 开发环境启动springboot(Intellij Idea)
```java
@SpringBootApplicatio
public class Main {
    public static void main(String[] args) throws IOException {
        System.out.println(System.getProperty("java.class.path"));
    }
}
```

## java -jar 形式springboot的启动流程
- `JarLauncher`才是真正的入口
    ```text
    //MANIFEST.MF
    Main-Class: org.springframework.boot.loader.JarLauncher // jvm识别的是这个
    Start-Class: org.yuhao.springcloud.order.Main
    //...
    ```

- 启动JarLauncher
    ```java
    // org.springframework.boot.loader.JarLauncher#main
    // 这是jar包真正的main
    public static void main(String[] args) throws Exception {
		new JarLauncher().launch(args);
	}
    //org.springframework.boot.loader.Launcher#launch(java.lang.String[])
    protected void launch(String[] args) throws Exception {
        JarFile.registerUrlProtocolHandler();
        // 创建一个新的classloader
        // 这个classload加载的路径就是 BOOT-INF/classes/ & BOOT-INF/lib/xxx.jar
        // 所以springboot解决这个问题的方式是自定义一个classloader 
        // 方然这个classloader也是URLClassloader
		ClassLoader classLoader = createClassLoader(getClassPathArchives());
        // getMainClass() 取的就是我们自定义的Main
		launch(args, getMainClass(), classLoader);
	}
    ```
- 自定义主类的启动
    ```java
    //org.springframework.boot.loader.Launcher#launch(java.lang.String[], java.lang.String, java.lang.ClassLoader)
    protected void launch(String[] args, String mainClass, ClassLoader classLoader) throws Exception {
		// 设置为当前线程类加载器
        Thread.currentThread().setContextClassLoader(classLoader);
		createMainMethodRunner(mainClass, args, classLoader).run();
	}
    //org.springframework.boot.loader.MainMethodRunner#run
    public void run() throws Exception {
		// 加载主类
        Class<?> mainClass = Thread.currentThread().getContextClassLoader().loadClass(this.mainClassName);
		// 反射启动主类
        Method mainMethod = mainClass.getDeclaredMethod("main", String[].class);
		mainMethod.invoke(null, new Object[] { this.args });
	}
    ```

- Idea启动时是Idea帮我们设置classpath，然后Appclassloader读classpath实现的
- java -jar 启动时是springboot的JarLauncher创建了一个特殊的加载器实现的
